Welcome to the official repository for the **RioRash** website. RioRash is a premium tech company providing cutting-edge solutions in Web, App, Game, and AI development. 

This website was built to reflect the high-tech, forward-thinking nature of the brand, utilizing a deeply immersive **Futuristic Cyber/Sci-Fi Design**.

## ✨ Features

- **High-Tech Custom Cursor**: A custom-built targeting reticle cursor that tracks the mouse and spins/glows when interacting with buttons.
- **Deep Space Neon Aesthetic**: A beautiful dark-mode palette featuring deep space blacks with blazing neon cyan and electric purple glows.
- **Advanced Interactivity**:
  - **3D Tactile Tilt**: Service and technology cards physically tilt in 3D space based on mouse position.
  - **Magnetic Buttons**: Call-to-action buttons subtly pull toward the user's cursor.
- **Futuristic Typography**: Uses Google Fonts `Orbitron` and `Space Grotesk` for a robotic, AI-driven feel.
- **Dynamic Aurora Backgrounds**: A breathing, animated gradient mesh in the hero section.
- **Responsive Layout**: Fully optimized for Desktop, Tablet, and Mobile devices with a custom mobile navigation menu.

## 🛠️ Technology Stack

This project is built using **Vanilla Web Technologies** to ensure maximum performance and seamless deployment without the need for complex build tools.

- **HTML5**: Semantic and accessible structure.
- **CSS3**: Advanced features including `backdrop-filter` (glassmorphism), CSS animations, CSS Variables, and Grid/Flexbox layouts.
- **JavaScript (ES6)**: Lightweight logic for intersection observers (scroll reveal), 3D tilt math, cursor tracking, and mobile menus.

## 🚀 Deployment (GitHub Pages)

This project is 100% ready to be deployed to GitHub Pages out-of-the-box. 

1. **Fork or Clone** this repository.
2. Push the files to your own GitHub repository.
3. Navigate to your repository's **Settings** > **Pages**.
4. Under "Build and deployment", select the `main` branch as your source.
5. Save, and your futuristic tech site will be live within minutes!
